# Citadel of Power Church Management System

A full-stack church management system with admin dashboard for managing workers, sermons, events, and contact messages.

## 🚀 Live Demo
- Frontend: https://citadel-of-power.vercel.app
- Admin Login: https://citadel-of-power.vercel.app/admin/login

## 📋 Features
- ✅ Public website with church information
- ✅ Admin dashboard for content management
- ✅ Workers directory with image uploads
- ✅ Sermons management with video URLs
- ✅ Events calendar with registration
- ✅ Contact form with admin panel
- ✅ Secure authentication

## 🛠️ Tech Stack
- **Frontend:** React, Vite, TailwindCSS, Framer Motion
- **Backend:** Node.js, Express, MongoDB
- **Storage:** Cloudinary for images
- **Deployment:** Vercel

## 📁 Project Structure
├── backend/ # Node.js/Express API
├── frontend/ # React/Vite application
├── .gitignore # Git ignore rules
└── README.md # Project documentation


## 🏁 Getting Started

### Prerequisites
- Node.js 18+
- MongoDB
- Cloudinary account

### Installation
1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Install frontend dependencies: `cd frontend && pnpm install`
4. Set up environment variables (see `.env.example` files)
5. Run development servers:
   - Backend: `cd backend && npm run dev`
   - Frontend: `cd frontend && pnpm run dev`
